Markdown

Markdown is a great way to format text for web without having to type out pesky HTML. Us developers use it constantly, and you'll grow to like it too. I promise.

Markdown is the best!

Galleries

Galleries: A list of all of the galleries along with their nested comments. This page is the most useful for leaving comments, because they're grouped by gallery upload.

Comments

Similar to the Galleries view, but the comments are not grouped. This will surface comments that have been left on old galleries that would not rise to the top of the Galleries view.

Images

Upload public images. This is great for adding images to student comments. Hide little-used images to get rid of distractions while maintaining the images in the system so that students can view them.

Announcements

Post announcments for logged-in users in this view. You can deactivate announcements rather than delete them, because you may want to repeat announcements in the future.

Discounts

Edit discount codes, but do it carefully. Users rely on these codes for checkout.

Transactions

Fulfill orders and watch transaction flow.

Files

Manage permissions for private server files. These guys are inaccessible to users who are not logged in.
